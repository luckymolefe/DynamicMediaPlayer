# `Dynamic Music Player`

- __Dynamic Music Player is simple music player uploads mp3 music to playlist and click track from the playlist to load into mediaplayer to autoplay.__

## `Built With`

- PHP
- HTML5
- CSS3
- JAVASCRIPT JQUERY
